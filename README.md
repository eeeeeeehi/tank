## 概要
**Neon Tank Arena** は、HTML5 Canvas と TypeScript で開発された **ローグライク要素を持つ 2D 戦車アクションシューティングゲーム** です。
サイバーパンクなネオン空間で、迫りくる敵戦車を撃破し、自機を強化（ビルド）しながら生き残りを目指します。

---

## 🎮 ゲームモード

### 🚀 Campaign Mode（キャンペーンモード）
- **ステージクリア型**
  - エリア内の敵を全滅させるとクリア。
  - **5ステージごとに強力なボス**が登場します！
- **成長システム（ローグライク）**
  - ステージクリア時に、ランダムに提示される「モジュール」を選択して自機を強化。
  - **スキルビルド形式**: 同じスキルを重ねて取ると、レベルが上がり性能が劇的に向上します。
    - 例: 「拡散弾モジュール」を重ねると、3方向 → 5方向 → 7方向と発射数が増加！

### 👑 Battle Royale（バトルロイヤル） [Bキー]
- **サバイバル戦**
  - プレイヤーを含む複数の戦車が入り乱れる大乱戦モード。
  - 最後の1人（Champion）になるまで生き残れ！

### 💀 Survival Mode（サバイバルモード） [Sキー]
- **無限耐久戦**
  - 四方八方から無限に出現する敵の波状攻撃をしのぎ、どれだけ長く生き残れるかを競います。
  - **時間経過で難易度上昇**: 生存時間が長くなるほど、敵が強力になり、数も増えます。
  - **ハイリスク仕様**: 被弾後の無敵時間が極端に短く（0.1秒）、囲まれると一瞬で撃破される緊張感があります。
  - **ランキング機能**: 生存時間と到達レベルが記録され、自己ベスト更新を目指せます。

## ✨ 主な特徴

### 🛠️ 深化したアップグレードシステム
従来の「武器持ち替え」ではなく、「積み上げ式」の強化システムを採用。
- **拡散弾モジュール (Shotgun)**: 同時に発射する弾数が増加。広範囲を制圧。
- **ラピッドファイア (Machinegun)**: 連射速度が向上。弾幕を張れる。
- **スナイパーモジュール (Sniper)**: 弾速と威力が向上。遠距離からの一撃。
- **ヘビィキャノン (Heavy)**: 破壊力抜群の大口径弾を発射。
- **AP弾 (Penetrating Shot)**: 弾丸が敵を貫通するようになり、集団戦で威力を発揮。

### 👹 ボスバトル & 敵AI
- **Boss Battle**: 5ステージごとに巨大ボスが出現。
  - ボスの放つ**巨大弾**は、こちらの弾をかき消して進んでくるため、回避が重要です。
  - 段階ごとにボスのレベルが上がり、攻撃が激化します。
- **賢い敵AI**:
  - スナイパーは遠距離を維持し、ショットガンは接近戦を挑むなど、役割に応じた立ち回りをします。
  - 自機の弾は**常に緑色**で見やすく、敵弾との識別が容易です。

### 🛡️ 快適なプレイ感
- **安全な初期配置**: スタート直後に敵と衝突しないよう、安全地帯から開始できます。
- **誤操作防止**: ステージクリア直後のスキル選択画面では、連打による誤選択を防ぐため、一瞬入力が無効化されます。
