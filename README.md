# Neon Tank Arena  
**2D Rogue-like Tank Action Shooter (HTML5 Canvas / TypeScript)**

このゲームは、**HTML5 Canvas と TypeScript** で開発された  
**ローグライク要素を持つ 2D 戦車アクションシューティングゲーム**です。

サイバーパンクなネオングリッド空間を舞台に、  
多様な敵戦車と戦いながら自機を強化し、生き残りを目指します。

---

## 🎮 ゲームモード

### 🚀 Campaign Mode（キャンペーンモード）
- **ステージクリア型**
  - 敵戦車をすべて撃破するとステージクリア
- **成長システム（ローグライク要素）**
  - ステージクリア時にランダムなアップグレードが3つ提示
  - その中から1つを選択して戦車を強化
- **アップグレード例**
  - Engine Boost（移動速度アップ）
  - Rapid Reloader（連射速度アップ）
  - Reinforced Armor（最大HPアップ＆回復）
  - Spread Shot Module（3方向ショットガン化）
- ステージが進むほど、敵の数と強さが増加

---

### 👑 Battle Royale（バトルロイヤル）
- **10人対戦**
  - プレイヤー1人 vs CPU 9体
- **ハードコアルール**
  - ライフは1つのみ
  - 敵同士も戦い合うカオスな戦場
- 最後の1人 **Champion** を目指す生存戦

---
